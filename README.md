# LoanCalc 💰

A free, instant EMI and mortgage calculator. No signup, no backend, no API keys — just pure HTML, CSS and JavaScript.

🔗 **Live Site:** https://loancalc-inr.vercel.app/

---

## Features

- 📊 Instant EMI calculation with amortization schedule
- 🌍 Multi-currency support — INR, USD, EUR, GBP, JPY
- 🍩 Visual donut chart showing principal vs interest breakdown
- 📅 Year-by-year payment schedule
- 📱 Fully responsive — works on mobile and desktop
- ⚡ Zero dependencies — no frameworks, no build tools

## How to Use

Just open `index.html` in any browser. That's it.

## Deployment

Deployed on **Vercel** as a static site. Any push to the `main` branch auto-deploys.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
